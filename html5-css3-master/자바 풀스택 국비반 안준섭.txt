CSS 쪽지 시험 2024-04-30
1. 스타일시트
2. 스타일 지정이 끝났다를 뜻한다
3. /*내용*/으로 표현한다
4. 내부 스타일 시트 : 파일 내 스타일 태그를 사용하여 지정한것
   외부 스타일 시트 : 별도의 css 파일을 사용하여 스타일을 작성 후
                             필요에 따라 불러와 사용한다
   인라인 스타일 시트 : 사용하는 태그에 style을 작성하여 사용한다
5. *{내용} 으로 표현한다
6. 클래스 선택자 : 호출 할때 '.클래스' 형식으로 사용한다.여러번 사용이
    가능하다
   아이디 선택자 : 호출 할때 '#클래스' 형식으로 사용한다. 한번만 사용이
   하다.
7. <div>는 지정한 전체를 묶어 적용할 때 사용하며 <span>은 텍스트 사이에
   스타일을 지정할 때 사용한다
8. 마우스를 올릴시에 숨겨진 텍스트를 보이게 해준다.
9. 둘이상  요소에 같은 스타일 적용 하려면 어떻게 고쳐야 하는가
: h1 h2 { text-align:center };
